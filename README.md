## 🧪 Test Automation

### 🔹 [TMS_v2 E2E Test Automation - Playwright](./projects/playwright/README.md)

**E2E 테스트 자동화 프로젝트**
- **기술 스택**: Playwright, JavaScript
- **테스트 대상**: [TMS_v2](https://tms-v2-phi.vercel.app/)
- **주요 기능**:
  - 로그인부터 테스트케이스 생성까지 전체 플로우 자동화
  - 로그인부터 플랜 생성까지 전체 플로우 자동화
  - 영상 녹화 및 스크린샷 자동 캡처
  - 상세한 테스트 리포트 생성 (HTML, JSON, JUnit)
  - 로그인 폼 영역에서만 에러 체크하는 정교한 에러 처리

**테스트 시나리오**

**플로우 1: 로그인 → 테스트케이스 생성**
1. 로그인 페이지 접속 및 자동 로그인
2. Test Cases 페이지 이동
3. "Add case" 버튼 클릭 및 모달 팝업
4. 테스트케이스 정보 입력 (Title, Precondition, Steps, Expected Result)
5. 저장 및 성공 확인

**플로우 2: 로그인 → 플랜 생성**
1. 로그인 페이지 접속 및 자동 로그인
2. Test Plans & Runs 페이지 이동
3. "플랜 생성" 버튼 클릭 및 플랜 생성 페이지 진입
4. 플랜 정보 입력 (플랜 이름, 설명)
5. 테스트케이스 선택 (목록에서 일부 선택)
6. 플랜 생성 및 플랜 목록 페이지 이동 확인
7. 활성 플랜 목록에서 생성한 플랜 클릭하여 상세 페이지 진입
