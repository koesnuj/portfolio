# 👋 Welcome to My QA Portfolio

안녕하세요. 저는 **QA 기반의 사고력과 생성형 AI 활용 능력**을 바탕으로  
서비스의 품질을 높이는 데 집중하는 **QA 임준석**입니다.

비전공자에서 출발했지만, 요구사항 분석·기능 설계·디버깅·테스트 설계 등  
QA의 핵심 역량을 기반으로 **AI 개발 도구(Claude, Gemini,Cursor 등)**를 활용해  
실제 테스트 관리리 시스템(TMS_v2)을 단독으로 개발하며 품질 중심의 사고를 확장했습니다.

## 🧩 My Key Strengths (QA 관점)

### ✔ 요구사항 명확화 능력  
- 기능 흐름, 예외 케이스, 사용자 시나리오를 기준으로 한 구체적 정의 능력  
- 추상적 요구사항을 테스트 가능한 형태로 구조화

### ✔ 테스트 설계 및 케이스 분해  
- 정상/비정상/경계값 테스트 설계  
- 흐름 기반 시나리오 테스트 작성 가능

### ✔ 문제 재현 & 디버깅 능력  
- 문제 상황을 분해하고 재현 조건을 빠르게 파악  
- 로그 기반 원인 분석  
- 기능 간 연계 오류 추적 능력

### ✔ AI 활용 개발 능력  
- 기능 정의를 통해 AI 개발 방향을 통제  
- 프롬프트 설계 및 개선 능력  
- AI의 한계를 분석하고 보완 전략 적용
